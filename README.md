
STM32智能药盒功能如下：
1.	基于STM32F103C8T6单片机主控
2.	监测功能：环境温湿度、药品余量、人体吃药状态
3.	缺药报警：药物重量为零时蜂鸣器报警提醒缺药
4.	时间显示：可以设置和显示当前的实时时间
5.	定时提醒：可以设置5个定时时间提醒吃药，每个时间可自由控制开关
6.	语音播报：定时吃药时间到扬声器播报语音提醒吃药
7.	APP控制：连接蓝牙APP接收环境信息并可设置定时时间与开关


2.材料清单

•	STM32F103C8T6单片机

•	OLED 屏幕

•	DHT11温湿度传感器

•	光电红外传感器

•	HX711称重模块

•	JR6001语音模块

•	BT04A蓝牙模块

•	有源蜂鸣器
